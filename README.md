# miniassembly-theme README

MiniAssembly is a Assembly language based of the Assembly LManguage created by Pr Ledent for the course "CI2 Licence Informatique Fondamentale Grands-Moulins - Université Paris Cité".
To execute MiniAss, first compile it with 
```
miniassc file.ass
``̀

then paste it to Pr Ledent website : https://www.irif.fr/~ledent/teaching/ci2/
for more info use (fr)
`̀``
miniassc -d
```

This VSCode extension provides Syntax Highlighting for *.ass files as well as auto completion tooltips
